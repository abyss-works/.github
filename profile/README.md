# abyss-works

Personal Agentic-DevOps playground — K8s cluster, self-hosted runners, and a handful of applications.

## Stack

| Layer | What |
|-------|------|
| Cluster | Kind (3-node) |
| Runner | Self-hosted GitHub Actions runner (Docker Compose) |
| Agent | Hermes AI agent (custom fork) |
